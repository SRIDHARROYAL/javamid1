1. Define class & object.

A class is a blueprint for creating objects, defining a set of properties and behaviors that the objects created from the class can have. An object is an instance of a class.

2. Syntax of creation of class and object.

The syntax for creating a class and an object is as follows:

class MyClass {
   int x;
   void display() {
     System.out.println(x);
   }
}
public class Main {
   public static void main(String[] args) {
     MyClass obj = new MyClass();
     obj.x = 10;
     obj.display();
   }
}
3. Define constructor.

A constructor is a special method in a class that is called when an object of that class is instantiated. It initializes the object.

class MyClass {
   int x;
   MyClass(int a) {
     x = a;
   }
   void display() {
     System.out.println(x);
   }
}
public class Main {
   public static void main(String[] args) {
     MyClass obj = new MyClass(5);
     obj.display();
   }
}
4. Difference between method & constructor.

A method performs operations and can be called multiple times, while a constructor initializes the object and is called only once during object creation. Constructors do not have a return type, while methods do.

5. What is the use of the "this" keyword?

The this keyword refers to the current instance of the class. It is used to differentiate instance variables from parameters with the same name and to invoke other constructors within the class.

class MyClass {
   int x;
   MyClass(int x) {
     this.x = x;
   }
   void display() {
     System.out.println(this.x);
   }
}
6. What is meant by nested classes?

Nested classes are classes defined within another class. They can be static or non-static. Non-static nested classes are called inner classes.

class OuterClass {
   class InnerClass {
     void display() {
       System.out.println("Inner class");
     }
   }
}
public class Main {
   public static void main(String[] args) {
     OuterClass outer = new OuterClass();
     OuterClass.InnerClass inner = outer.new InnerClass();
     inner.display();
   }
}

Static is used for class-level variables and methods that are shared among all instances.
Final defines constants and restricts method overriding and inheritance.

class MyClass {
   static int count = 0;

   void increment() {
     count++;

8. List out the access specifiers.

Access specifiers determine the visibility of class members:
- private: Accessible only within the class.
- default: Accessible within the same package.
- protected: Accessible within the same package and subclasses.
- public: Accessible from anywhere.

9. What is method overriding?

Method overriding allows a subclass to provide a specific implementation for a method that is already defined in its superclass. The method signature in the subclass must be the same as in the superclass.

class Parent {
   void show() {
     System.out.println("Parent class");
   }
}
class Child extends Parent {
   @Override
   void show() {
     System.out.println("Child class");
   }
}
public class Main {
   public static void main(String[] args) {
     Parent obj = new Child();
     obj.show(); // Output: Child class
   }
}
10. Difference between class & constructor.

A class defines the structure and behavior of objects, while a constructor initializes the objects of the class. The class is a blueprint, whereas the constructor is a special method used for object creation.

11. Explain the types of polymorphism.

Polymorphism allows objects to be treated as instances of their parent class. Types include:
- Compile-time Polymorphism (Method Overloading): Multiple methods with the same name but different parameters.
- Runtime Polymorphism (Method Overriding): Overriding methods in a subclass to provide a specific implementation.

12. Explain compile-time polymorphism (method overloading).

Compile-time polymorphism or method overloading allows a class to have more than one method with the same name, but with different parameters (type, number, or both).

class MathOperation {
   int add(int a, int b) {
     return a + b;
   }
   double add(double a, double b) {
     return a + b;
   }
}
public class Main {
   public static void main(String[] args) {
     MathOperation op = new MathOperation();
     System.out.println(op.add(5, 10)); // Output: 15
     System.out.println(op.add(5.5, 10.5)); // Output: 16.0
   }
}
13. Explain runtime polymorphism (method overriding).

Runtime polymorphism or method overriding allows a subclass to provide a specific implementation of a method that is already defined in its superclass. The method in the subclass must have the same name, return type, and parameters as in the superclass.

class Animal {
   void makeSound() {
     System.out.println("Some sound");
   }
}
class Dog extends Animal {
   @Override
   void makeSound() {
     System.out.println("Bark");
   }
}
public class Main {
   public static void main(String[] args) {
     Animal a = new Dog();
     a.makeSound(); // Output: Bark
   }
}
14. Explain passing arguments by value and reference.

Pass by Value: A copy of the value is passed to the method. Changes to the parameter do not affect the original value.
Pass by Reference: The reference to the object is passed. Changes to the object affect the original object.

class PassByValue {
   void modify(int x) {
     x = 10;
   }
}
class PassByReference {
   void modify(int[] arr) {
     arr[0] = 10;
   }
}
public class Main {
   public static void main(String[] args) {
     PassByValue obj1 = new PassByValue();
     int num = 5;
     obj1.modify(num);
     System.out.println(num); // Output: 5
     PassByReference obj2 = new PassByReference();
     int[] numbers = {5};
     obj2.modify(numbers);
     System.out.println(numbers[0]); // Output: 10
   }
}
15. How to assign one object to another.

To assign one object to another, simply assign the reference of one object to another variable. Both variables will refer to the same object.

class MyClass {
   int x;
   void display() {
     System.out.println(x);
   }
}
public class Main {
   public static void main(String[] args) {
     MyClass obj1 = new MyClass();
     obj1.x = 10;
     MyClass obj2 = obj1;
     obj2.display(); // Output: 10
   }
}

20. What is typecasting & type conversion/widening?

Typecasting is converting one data type to another, either explicitly or implicitly.
Type conversion/widening is the automatic conversion of a smaller data type to a larger data type.

class Main {
   public static void main(String[] args) {
     int i = 10;
     double d = i; // Widening conversion
     System.out.println(d); // Output: 10.0
     double num = 10.5;
     int n = (int) num; // Narrowing conversion
     System.out.println(n); // Output: 10
   }
}
21. Bitwise operators.

Bitwise operators perform operations on the binary representations of integers. They include:
- `&`: AND
- `|`: OR
- `^`: XOR
- `~`: NOT
- `<<`: Left shift
- `>>`: Right shift
- `>>>`: Unsigned right shift

class Main {
   public static void main(String[] args) {
     int a = 5; // 0101
     int b = 3; // 0011
     System.out.println(a & b); // Output: 1
     System.out.println(a | b); // Output: 7
     System.out.println(a ^ b); // Output: 6
     System.out.println(~a); // Output: -6
   }
}
22. What is the ternary/conditional operator?

The ternary operator is a shorthand for the `if-else` statement. It has the form:
condition ? value_if_true : value_if_false

class Main {

     String result = (a > 5) ? "Greater" : "Smaller";

   }
}

24. Write about break and continue keywords.

Break terminates the nearest enclosing loop or switch statement.
Continue skips the current iteration of the loop and proceeds to the next iteration.

class Main {
   public static void main(String[] args) {
     for (int i = 0; i < 5; i++) {
       if (i == 2) break;
       System.out.println(i);
     }
     for (int i = 0; i < 5; i++) {
       if (i == 2) continue;
       System.out.println(i);
     }
   }
}
25. Difference between '==' and 'equals()' method.

== compares references to check if two objects refer to the same instance.
equals() compares the actual content of the objects. For custom objects, you need to override the `equals()` method to define what equality means.

class MyClass {
   int x;
   MyClass(int x) {
     this.x = x;
   }
   @Override
   public boolean equals(Object obj) {
     if (this == obj) return true;
     if (obj == null || getClass() != obj.getClass()) return false;
     MyClass myClass = (MyClass) obj;
     return x == myClass.x;
   }
}
public class Main {
   public static void main(String[] args) {
     MyClass obj1 = new MyClass(5);
     MyClass obj2 = new MyClass(5);
     System.out.println(obj1 == obj2); // Output: false
     System.out.println(obj1.equals(obj2)); // Output: true
   }
}
26. Write about try, catch and finally blocks.

Try: Contains code that might throw an exception.
Catch: Handles the exception if it occurs.
Finally: Contains code that will execute regardless of whether an exception occurs or not.

class Main {
   public static void main(String[] args) {
     try {
       int result = 10 / 0;
     } catch (ArithmeticException e) {
       System.out.println("Cannot divide by zero");
     } finally {
       System.out.println("This block always executes");
   }
   }
}

30. What is recursion? Write an example program to print the factorial of a given number.

Recursion is a technique where a method calls itself to solve a problem. It's useful for problems that can be divided into similar sub-problems.

class Factorial {
   static int factorial(int n) {
     if (n == 0) return 1;
     else return n * factorial(n - 1);
   }
   public static void main(String[] args) {
     int result = factorial(5);
     System.out.println(result); // Output: 120
   }
}
31. What is meant by method overloading? Explain how the 'this' keyword is used in method overloading with an example program.

Method overloading is defining multiple methods with the same name but different parameters. The this keyword can be used to call one constructor from another within the same class.

class OverloadingExample {
   void display(int a) {
     System.out.println("Integer: " + a);
   }
   void display(String a) {
     System.out.println("String: " + a);
   }
   void display(int a, String b) {
     System.out.println("Integer and String: " + a + ", " + b);
   }
   public static void main(String[] args) {
     OverloadingExample obj = new OverloadingExample();
     obj.display(10);
     obj.display("Hello");
     obj.display(10, "World");
   }
}

33. What is the difference between C and Java?

C is a procedural programming language, while Java is an object-oriented programming language. Java includes features like automatic garbage collection, exception handling, and a robust standard library, which are not present in C. Java is platform-independent, whereas C is platform-dependent.

35. What is type conversion and type casting with a suitable example?

Type conversion (or widening) is automatically converting a smaller type to a larger type. Type casting (or narrowing) is explicitly converting a larger type to a smaller type.

class TypeConversion {
   public static void main(String[] args) {
     // Type Conversion
     int i = 10;
     double d = i; // Automatic conversion
     System.out.println(d); // Output: 10.0
     // Type Casting
     double e = 9.8;
     int f = (int) e; // Explicit conversion
     System.out.println(f); // Output: 9
   }
}
